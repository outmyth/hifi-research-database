# Changelog

All notable updates to Vincent HiFi Research Database (VHRD) will be documented here.

## 2026-06-25

### Added

- Initialized the repository with `README.md`.
- Added project roadmap and core maintenance files.
- Added first version of the Vincent Top 20 collector list.
- Added active watchlist and wishlist.
- Added reusable product model template.
- Added `INDEX.md` as the main navigation page.
- Added `TODO.md` for future listening notes, market tracking, and missing model pages.
- Added system and portable reference overview pages.
- Added buying and cable/adapter workflows.
- Added initial Audeze model files: LCD-2, LCD-2C, LCD-3, LCD-X, LCD-5s.
- Added Koss/Stax electrostatic files: ESP/950, ESP/95X, SR-007 Mk1, SR-Lambda Normal Bias, SR-404.
- Added Grado/Sony classic dynamic files: RS1 Vintage, PS500e, GH2, HP1000, MDR-CD3000.
- Added main owned-system files: HD800, HDVD 800, Luxman P-1u, Sony HAP-Z1ES, WM1ZM2, Chord Hugo, Chord Mojo + Poly, Chord Qutest + Anni, LPG 2017, SP1000 Copper, ACRO CA1000, Plenue S, FangSound Ariadne, Woo WA6, EF400, Dethonray Listening M1, Utsusemi Enthusiast.
- Added IEM files: IE900, DUNU Falcon Ultra, SeeAudio Bravery, QoA Aviation, Sound Rhyme SM2, Astrotec AM850MK2, Technics TZ700.
- Added comparison files for Audeze, Koss/Stax, Grado, Sony, metal listening, and IE900 vs Falcon Ultra.

### Notes

This database is intended to be updated whenever a listing, model comparison, market datapoint, ownership note, listening impression, cable pinout, or seller message becomes relevant.
