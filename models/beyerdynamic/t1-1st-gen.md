# Beyerdynamic T1 1st Generation

## Summary

A classic high-impedance Beyerdynamic dynamic headphone in Vincent's collection. It should be used as the high-impedance dynamic reference and a pairing test for Luxman P-1u, Woo Audio WA6, and other amplifiers.

## Status

- Category: Open-back dynamic headphone
- Collector tier: S
- Current ownership: Owned
- Priority: Keep as high-impedance dynamic reference

## Sound Role

- High-impedance dynamic reference.
- Useful for testing tube amps and refined solid-state amps.
- More focused than HD800, but still analytical/bright-leaning depending on chain.

## System Synergy

- Luxman P-1u: likely refined/smooth pairing candidate.
- Woo Audio WA6: important tube pairing candidate.
- HDVD 800: compare as clean solid-state baseline.
