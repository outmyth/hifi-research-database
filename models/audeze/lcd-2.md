# Audeze LCD-2

## Summary

A foundational Audeze planar classic. For Vincent, LCD-2 is interesting when it is an older desirable revision or a very strong value deal, but LCD-3 remains the higher-priority long-term Audeze target.

## Status

- Category: Open-back planar magnetic headphone
- Collector tier: S
- Current ownership: Target only at strong price
- Priority: Medium-high depending on version and price

## Preferred Version

- LCD-2 Rev.2 is the most interesting collector/reference version for this database.
- Later Fazor and modern LCD-2 versions can still be good, but should be priced lower than LCD-3 targets.
- Carbon fiber headband is a comfort bonus, not the main reason to buy.

## Sound Signature

### Bass

Weighty, rounded, and classic Audeze in character. Not as technically controlled as modern LCD-X.

### Midrange

Warm and rich, but generally not as luxurious or refined as LCD-3.

### Treble

Smooth and relaxed. Some versions can sound dark.

### Soundstage / Imaging

More intimate than HD800. Good planar presence, not a huge-stage headphone.

### Dynamics / Speed

Satisfying, but not as fast or studio-precise as LCD-X.

## System Synergy

- Complements HD800 by adding warmth and body.
- Overlaps with LCD-3 target, so price must justify purchase.
- Good for relaxed rock and atmospheric listening.

## Head-Fi Consensus

### Common Praise

- Classic Audeze warmth
- Thick tone
- Smooth treble
- Strong used-market value when priced well

### Common Criticism

- Heavy
- Can sound dark
- Less technical than LCD-X and LCD-3
- Version differences matter

## Reliability / Known Issues

- Confirm driver health and channel balance.
- Check pad condition.
- Check wood cups for cracks.
- Ask about repair/service history.

## Buying Guide

### Must Check

- Exact revision
- Fazor vs non-Fazor
- Driver condition
- Wood condition
- Pads/headband
- Included cable and case

### Nice to Have

- Rev.2 documentation
- Original case
- Upgraded headband
- Recent pads

### Avoid

- Overpriced modern LCD-2 near LCD-3 pricing
- Cracked cups
- Driver imbalance
- Vague version claims
