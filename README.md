# HiFi Research Database (HRD)

A personal HiFi research database for tracking classic headphones, DACs, amplifiers, market prices, buying notes, and long-term collector value.

This project is designed as a living Markdown knowledge base rather than a static ranking list. It focuses on:

- Used-market price intelligence
- Version identification
- Head-Fi / community consensus
- Reliability and maintenance notes
- Collector value
- System synergy with Vincent's current gear
- Genre fit, especially metal, progressive rock, post-punk, jazz, and classic rock

## Core Workflow

When evaluating a listing, update the relevant model file and market log:

1. Identify exact version and condition.
2. Compare current asking price with HifiShark, HifiZero, Head-Fi, USAudioMart, eBay sold listings, and other available market references.
3. Check known issues and repair risk.
4. Score against Vincent's existing system and music taste.
5. Add the listing to `market/` or `watchlist.md` when useful.

## Scoring Philosophy

The database is not a generic audiophile ranking. It prioritizes:

- Classic status over hype
- Long-term used-market value over MSRP
- Unique sonic character over minor technical upgrades
- Repairability and ownership risk
- Fit with Vincent's actual music and system
