# Vincent System Reference

This page summarizes Vincent's current reference gear so that every future headphone, IEM, DAC, and amplifier evaluation can be judged against the actual system rather than in isolation.

## Desktop Headphone Reference

### Sennheiser HD800

Role:

- Soundstage and imaging reference
- Progressive metal / jazz / atmospheric rock benchmark
- Treble and pad-condition sensitivity reference

Primary pairings:

- Sennheiser HDVD 800
- Luxman P-1u
- Sony HAP-Z1ES as source
- Chord Qutest + Anni stack

### Luxman P-1u

Role:

- Refined desktop amplifier reference
- Potentially warm/smooth counterbalance for HD800 and Beyerdynamic T1
- Main amp baseline for high-end dynamic and planar headphone pairing notes

### Sennheiser HDVD 800

Role:

- Native HD800-era DAC/amp reference
- Clean and convenient Sennheiser desktop baseline
- Useful contrast against Luxman P-1u

## Evaluation Rules

When evaluating a new item, always ask:

1. Does it add a genuinely new sound signature?
2. Does it duplicate HD800, LCD-3 target, or existing Grado roles?
3. Is the used price close to dream/great-deal range?
4. Does it fit Vincent's core genres?
5. Is reliability acceptable?
6. Does it have collector or long-term value?
