# Buying Evaluation Workflow

Use this workflow whenever Vincent sends a listing, photo, price, seller message, or marketplace link.

## Step 1: Identify Version

- Exact model and revision
- Production year / serial clues
- Accessories and cable type
- Known service or repair history
- Photos that confirm or contradict seller claims

## Step 2: Check Market

Use available sources when current pricing matters:

- HifiShark
- HifiZero
- Head-Fi Classifieds
- USAudioMart
- Audiogon
- eBay sold listings
- Facebook Marketplace / local listings

Record price into the appropriate `market/YYYY-qX.md` file if useful.

## Step 3: Condition Risk

Check:

- Driver/channel balance
- Pad/headband wear
- Cable and connector condition
- Wood cup cracks or finish issues
- Battery health for DAPs
- Voltage and transformer needs for Japanese gear
- Electrostatic bias/energizer compatibility

## Step 4: System Fit

Compare against:

- HD800 reference role
- Luxman P-1u desktop amp role
- Sony WM1ZM2 portable warmth
- Chord technical reference
- Existing Grado/Audeze/Koss/Stax roles

## Step 5: Vincent Score

Score using:

- Sound: 30
- Collector value: 20
- Market price: 20
- Reliability: 10
- Metal fit: 10
- System synergy: 10

## Step 6: Decision

Use one of:

- S+ / Buy immediately
- A / Good buy
- B / Negotiate
- C / Pass unless special
- D / Avoid

## Step 7: Update Database

If relevant:

- Update model file
- Update comparison file
- Update market timeline
- Update watchlist/wishlist
- Add seller questions or negotiation notes
