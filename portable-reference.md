# Portable Reference

This page summarizes Vincent's portable source and IEM reference system.

## Main Portable Sources

### Sony WM1ZM2

Role:

- Main warm premium portable reference
- 4.4 mm balanced baseline
- First source to test when an IEM sounds thin, metallic, or too sharp

### Lotoo PAW Gold 2017

Role:

- Classic 3.5 mm single-ended reference
- Fast, direct, old-school flagship DAP flavor
- Important contrast against the warmer Sony WM1ZM2

### Chord Hugo

Role:

- Transportable Chord technical reference
- Useful for clarity, timing, and DAC/amp precision checks

### Cowon Plenue S

Role:

- Older DAP / secondary flavor
- Resale and collector note rather than main current reference

## Portable Evaluation Rules

When evaluating an IEM, always test or consider:

1. Sony WM1ZM2 for warmth and 4.4 mm balanced output.
2. LPG 2017 for old-school directness and single-ended reference.
3. Chord Hugo for technical clarity.
4. Tip synergy, especially if upper mids/treble or bass seal changes.
5. Whether it adds a new role beyond IE900 and Falcon Ultra.

## Core Portable Sound Goals

- Avoid thin or metallic sound for long metal sessions.
- Preserve fast attack for technical death metal and progressive metal.
- Keep enough bass weight for drums and electric bass.
- Avoid excessive upper-mid glare for melodic death metal guitars.
- Maintain vocal and jazz usability.
